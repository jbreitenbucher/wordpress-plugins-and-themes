Easily embed Stream videos in both Gutenberg and Classic Editor.

== Description ==

This plugin adds a Stream Embed Block to the Block Editor and a Stream oEmbed support for the Classic Editor.

Copy-paste the Stream video URL in the editor or use the dedicated Gutenberg block for more setting options.

1. Stream embed video.

== Installation ==

1. Install the plugin and activate it.
2. Copy-paste the Stream video URL in the editor or use the dedicated Gutenberg block for more setting options.
