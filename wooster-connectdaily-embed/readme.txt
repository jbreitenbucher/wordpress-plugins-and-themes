Add support for auto embedding Connect Daily calendars.

== Description ==

This plugin will allow you to embed a Connect Daily calendar just by pasting the link into your post or page.

== Installation ==

1. Upload the `wooster-connectdaily-embed` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Enter your settings in the Wooster Connect Daily options page.
1. Paste a link to a resource id (e. g., https://connectdaily.wooster.edu/ViewCal.html?dropdown=1&resource_id=279) into a post or page
